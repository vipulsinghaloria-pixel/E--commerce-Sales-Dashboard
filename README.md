 # PROJECT SUMMARY- 
"An interactive E‑Commerce Sales Dashboard built in Power BI, summarizing key performance metrics like total sales, profit, quantity, and average order value. It visualizes trends across months, states, categories, payment modes, and sub‑categories using bar charts and pie charts, enabling quick insights into regional performance, product profitability, and customer behavior."
